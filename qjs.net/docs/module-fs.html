fs (sync)

同步文件系统模块。在调用线程上阻塞执行；读写大文件请用 Stream 流式入口。

启用

new QuickJSEngine(opt => {
    opt.FileSystem         = true;
    opt.FileSystemBasePath = @"C:\sandbox";
});

所有相对路径都解析在 FileSystemBasePath 下；为空时使用进程当前目录。

读取

readFile(path, encoding?: string): string
readFileBytes(path): ArrayBuffer

写入

writeFile(path, content: string, encoding?: string): boolean
writeFileBytes(path, buffer: Buffer | ArrayBufferView | ArrayBuffer): boolean
appendFile(path, content: string, encoding?: string): boolean

流式入口 流

openRead(path): Stream
openWrite(path, append?: boolean): Stream
openDir(path): DirectoryStream

返回的 Stream / DirectoryStream 由引擎登记，未关闭时引擎销毁会兜底关闭。

查询

exists(path): boolean
isFile(path): boolean
isDirectory(path): boolean
stat(path): { size, isFile, isDirectory, created, modified, accessed } | null
readDir(path): string[]

操作

remove(path): boolean
rename(oldPath, newPath): boolean
copy(src, dst): boolean
mkdir(path): boolean
getcwd(): string
tempDir(): string
