fsAsync

异步文件系统模块；所有方法返回 Promise，由内置 EventLoop 调度。

启用

new QuickJSEngine(opt => {
    opt.AsyncFileSystem    = true;
    opt.FileSystemBasePath = @"C:\sandbox";
});

接口

除返回 Promise 之外，签名与 fs 一一对应，包含同样的查询/操作/读写/流入口：

  • readFile / readFileBytes
  • writeFile / writeFileBytes / appendFile
  • exists / isFile / isDirectory / stat
  • readDir
  • remove / rename / copy / mkdir
  • getcwd / tempDir
  • openRead(path): Promise<Stream>
  • openWrite(path, append?): Promise<Stream>
  • openDir(path): Promise<DirectoryStream>
  • getOpenFileName / getSaveFileName（Windows 对话框；可选）

示例

// 复制大文件，零跨语言往返
var src = await fsAsync.openRead('big-input.bin');
var dst = await fsAsync.openWrite('big-output.bin');
var n   = await src.pipe(dst);    // dst 自动关闭
await src.close();
console.log('copied', n, 'bytes');

注意

注意所有 fsAsync 调用必须包在 engine.Execute(...) 中；Eval 不会 pump 事件循环。
