TextEncoder / TextDecoder

最小可用版的 WHATWG 文本编解码器（仅 UTF-8）。

TextEncoder

new TextEncoder()
readonly encoding: "utf-8"
encode(str?: string): Uint8Array

TextDecoder

new TextDecoder(label?: string)
readonly encoding: string
decode(buffer?: ArrayBuffer | ArrayBufferView): string

启用

engine = new QuickJSEngine(opt => opt.Encoder = true);

与 Buffer 的对比

TextEncoder/Decoder 适合一次性转换；如果需要可变缓冲、按字节读写、流式处理，请用 Buffer（其 toString('utf8') / Buffer.from(str) 同样涵盖 UTF-8 互转）。
