Timers

setTimeout · setInterval · clearTimeout · clearInterval · queueMicrotask。

API

setTimeout(fn: () => void, delayMs: number): number
setInterval(fn: () => void, periodMs: number): number
clearTimeout(id: number): void
clearInterval(id: number): void
queueMicrotask(fn: () => void): void

语义

  • 所有定时器由 EventLoop 在调用 Execute 期间推进；不存在后台线程暗自触发回调。
  • 没有等待中的回调时 Execute 立即返回；之后的 setTimeout 回调不会被触发，除非再次 Execute。
  • microtask（来自 Promise / queueMicrotask）总是优先于宏任务（定时器、Task continuation）执行。

示例

engine.Execute(@"
    let i = 0;
    let id = setInterval(() => {
        if (++i >= 3) { clearInterval(id); }
        console.log('tick', i);
    }, 50);
    await new Promise(r => setTimeout(r, 200));
");
