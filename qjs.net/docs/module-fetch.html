fetch

基于 .NET HttpClient 的 fetch；返回 FetchResponse。

启用与配置

new QuickJSEngine(opt => {
    opt.Fetch        = true;
    opt.FetchOptions = new FetchModuleOptions {
        DefaultTimeout    = TimeSpan.FromSeconds(30),
        DefaultProxy      = "http://127.0.0.1:7890",
        IgnoreSslErrors   = false,
        DefaultUserAgent  = "qjs.net/1.0",
    };
});

签名

fetch(url: string, options?: FetchOptions): Promise<FetchResponse>

FetchOptions 字段：method · headers · body · timeout(ms) · proxy · ignoreSslErrors。

body 接受 string 或二进制（Buffer / Uint8Array / ArrayBuffer）。

FetchResponse 接口

ok: boolean · status: number · statusText: string · headers: Record<string, string> · url: string
text(): Promise<string>
json(): Promise<any>
arrayBuffer(): Promise<ArrayBuffer>
body(): Stream
互斥text() / json() / arrayBuffer() / body() 共享同一个底层响应——首次调用任意一个即消耗响应，后续调用其它任何一个都会抛错。
body() 返回的 Stream 拥有底层 HttpResponseMessage，关闭它即关闭整个响应（亦由引擎兜底）。

示例：下载到文件（流式）

var r = await fetch('https://example.com/file.zip');
if (!r.ok) throw new Error(r.status + ' ' + r.statusText);
var n = await r.body().pipe(fs.openWrite('file.zip'));
console.log('wrote', n, 'bytes');

示例：POST JSON

var r = await fetch('https://api.example.com/items', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ name: 'qjs', year: 2025 }),
});
var data = await r.json();
