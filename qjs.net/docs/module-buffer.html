Buffer

Node 风格的可变字节缓冲区。Memory<byte> 背书；slice 共享底层；toUint8Array 拷贝。

设计

  • 始终在引擎中可用（无需 opt-in）。
  • 底层 Memory<byte>；slice(start, end) 返回共享底层的子视图，不复制。
  • 作为 C# byte[] 形参传入时（fs/fetch 等），生成器透过 BufferUnwrapHook 自动 unwrap 出独立副本。
  • 不支持 buf[i] 索引，请用 get(i) / set(i, v)。

静态方法 static

Buffer.alloc(size: number, fill?: number): Buffer

分配 size 字节；fill 缺省 0。

Buffer.from(source, encoding?: string): Buffer

source 可为 string / ArrayBuffer / Uint8Array / 另一个 Buffer。字符串支持的 encoding：utf8/utf-8（默认）、ascii、latin1/binary、utf16/utf-16le、base64、hex。

Buffer.concat(list: Buffer[], totalLength?: number): Buffer

拼接；totalLength 缺省/<=0 时按各段长度求和；超出截断，不足补 0。

实例方法

length: number
get(i: number): number
set(i: number, v: number): void
slice(start?: number, end?: number): Buffer

返回共享底层的子视图。

copy(target, targetStart?, sourceStart?, sourceEnd?): number
fill(value: number, start?: number, end?: number): void
equals(other: Buffer): boolean
indexOf(value: number, from?: number): number
toString(encoding?: string): string

同 from() 编码集合；缺省 utf8。

toArrayBuffer(): ArrayBuffer
toUint8Array(): ArrayBuffer

都会拷贝出新的 ArrayBuffer。

数值读写

readUInt8 / readInt8 / readUInt16LE / readUInt16BE / readInt32LE / readInt32BE / readFloat32LE / readFloat64LE，对应同名 writeXxx(value, offset?)。

示例

var a = Buffer.from('hello, ');
var b = Buffer.from('world');
var c = Buffer.concat([a, b]);
c.toString();                    // "hello, world"
c.slice(7).toString();           // "world" — 共享底层内存

// 与原生 IO 互操作
fs.writeFileBytes('/tmp/x.bin', c);
var back = fs.readFileBytes('/tmp/x.bin');  // ArrayBuffer
Buffer.from(back).toString();    // "hello, world"

// 数值打包
var hdr = Buffer.alloc(8);
hdr.writeInt32LE(0xCAFEBABE | 0, 0);
hdr.writeInt32LE(c.length, 4);
hdr.toString('hex');             // e.g. "bebafeca0c000000"

生命周期

Buffer 是托管对象，由 GC 回收。无须手动释放。slice 返回的视图持有原 buffer 的引用，原 buffer 不会先于视图被回收。
