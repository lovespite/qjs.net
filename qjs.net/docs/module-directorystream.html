DirectoryStream

按批枚举目录的流式入口，比一次性 readDir 更适合超大目录。

设计

包装一个 IEnumerator<string>（来自 Directory.EnumerateFileSystemEntries），一次返回一批 DirEntry，直到 read 返回 null。与 Stream 一样登记到引擎，未关闭时引擎销毁会自动释放。

创建

fs.openDir(path: string): DirectoryStream
fsAsync.openDir(path: string): Promise<DirectoryStream>

方法

read(maxEntries?: number): Promise<DirEntry[] | null> async

取下一批最多 maxEntries 个条目；返回 null 表示已枚举完毕。maxEntries 缺省/<=0 时按内部默认（64）批量返回。

close(): Promise<void> async

DirEntry 结构

{
  name: string,         // 文件/目录名（不含父路径）
  isFile: boolean,
  isDirectory: boolean,
  size: number          // 对目录始终为 0
}

示例

var ds = await fsAsync.openDir('.');
try {
    while (true) {
        var batch = await ds.read(64);
        if (batch === null) break;
        for (var e of batch) {
            if (e.isFile) console.log(e.name, e.size);
        }
    }
} finally {
    await ds.close();
}
